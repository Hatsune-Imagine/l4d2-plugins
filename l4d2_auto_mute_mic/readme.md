# 自动闭麦



可配置玩家麦克风语音自动闭麦。

配置项：

```c
// 自动禁声自由麦.
// 0=关闭
// 1=开启
// -
// Default: "1"
l4d2_auto_mute_vox_mic "1"

// 自动禁声长时间麦.
// 0=关闭
// 1=开启
// -
// Default: "1"
l4d2_auto_mute_enable "1"

// 当玩家持续开麦达到几秒后, 自动禁声.
// -
// Default: "15.0"
l4d2_auto_mute_time "15.0"

// 禁声几秒后自动解禁.
// -
// Default: "5.0"
l4d2_auto_unmute_time "5.0"
```
